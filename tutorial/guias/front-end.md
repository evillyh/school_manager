← [Voltar ao README geral](../../README.md) · [Índice do tutorial](../README.md)

## Objetivo
Construir telas simples, claras e **funcionais** que consumam a API e cubram o fluxo do MVP.

## O que fazemos no MVP
- Login e guarda de rotas por perfil
- CRUD de **Alunos** e **Turmas**
- **Matrícula** (sem duplicar)
- **Lançamento de notas** e **Boletim** (visualização)

## Entregáveis deste bimestre
- [ ] Telas responsivas (desktop/celular)
- [ ] Estados de carregando/erro
- [ ] Formulários com validação básica
- [ ] Arquivo `.env.example` com `VITE_API_URL`

## Definition of Done (FE)
- Tela navega e salva/consulta na API
- Mensagens de erro claras
- Sem console errors
- Passou no **smoke** do QA

## Rotina da semana
1. Confirmar com PM o **contrato da API** (link)  
2. Implementar tela → conectar API → testar  
3. Entregar e revisar com QA

## Ferramentas
- React (Vite), React Router, Axios
